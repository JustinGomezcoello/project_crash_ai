3. Arquitectura recomendada del sistema
La arquitectura sugerida mantiene exactamente la logica de la Parte I, pero la aterriza a una
implementacion factible con herramientas accesibles. La idea no es construir un producto final industrial
desde el inicio, sino un prototipo inicial bien argumentado y extensible.
Bloque 1. Ingreso y normalizacion del video
Este bloque abre el archivo de video, lee metadatos como FPS, ancho y alto, y normaliza el frame de
trabajo. Debe conservar la relacion de aspecto, evitar deformaciones y permitir videos horizontales,
verticales o invertidos.
Bloque 2. Deteccion de objetos
Aqui se emplea YOLOv8 para detectar vehiculos. En el prototipo inicial conviene limitarse a clases
relevantes: car, motorcycle, bus y truck. Si luego se desea, se pueden incluir peatones u otros elementos.
Bloque 3. Tracking temporal
El objetivo es que cada vehiculo conserve una identidad entre frames. Eso permite estudiar trayectorias,
velocidad aparente, cambios bruscos de movimiento y posibles contactos entre pares de vehiculos.
Bloque 4. Inferencia de colision
Es el corazon del proyecto. No debe depender solo de una caja detectada, sino de una combinacion de
criterios: cercania, IoU, persistencia, cambio angular, perdida repentina de velocidad y consistencia en
varios frames.
Bloque 5. Salida explicable
El sistema debe devolver un video anotado con cuadros delimitadores, IDs, timestamp, banner de alerta y,
adicionalmente, un archivo resumen con frame del evento, segundo estimado e IDs involucrados.
3.1 Flujo tecnico completo
Video de entrada
-> lectura y validacion
-> normalizacion de frames
-> deteccion de vehiculos
-> tracking por ID
-> calculo de senales temporales
-> decision de choque
-> anotacion visual
-> exportacion de video y resumen

(cid:127) Facilita experimentar: se puede cambiar el detector, el tracker o la logica de colision sin rehacer todo.
(cid:127) Permite que otra IA continue el desarrollo porque la separacion entre bloques esta clara.

import cv2
import numpy as np
def resize_keep_aspect(frame, max_width=960):
h, w = frame.shape[:2]
if w <= max_width:
return frame
scale = max_width / float(w)
return cv2.resize(frame, (int(w * scale), int(h * scale)))
def box_center(box):
x1, y1, x2, y2 = box
return ((x1 + x2) / 2.0, (y1 + y2) / 2.0)
def box_diag(box):
x1, y1, x2, y2 = box
w = max(1.0, x2 - x1)
h = max(1.0, y2 - y1)
return float(np.sqrt(w * w + h * h))
def compute_iou(box_a, box_b):
ax1, ay1, ax2, ay2 = box_a
bx1, by1, bx2, by2 = box_b
inter_x1 = max(ax1, bx1)
inter_y1 = max(ay1, by1)
inter_x2 = min(ax2, bx2)
inter_y2 = min(ay2, by2)
inter_w = max(0.0, inter_x2 - inter_x1)
inter_h = max(0.0, inter_y2 - inter_y1)
inter_area = inter_w * inter_h
area_a = max(0.0, ax2 - ax1) * max(0.0, ay2 - ay1)
area_b = max(0.0, bx2 - bx1) * max(0.0, by2 - by1)

5.3 Archivo collision_logic.py
from collections import defaultdict, deque
import numpy as np
from utils import box_center, box_diag, compute_iou
class CollisionDetector:
def __init__(self, cfg):
self.cfg = cfg
self.track_history = defaultdict(lambda: deque(maxlen=cfg.history_size))
self.pair_streak = defaultdict(int)
self.events = []
def update(self, frame_idx, tracks, fps):
for track in tracks:
self.track_history[track["track_id"]].append(
box_center(track["bbox"])
)
new_events = []
for i in range(len(tracks)):
for j in range(i + 1, len(tracks)):
t1, t2 = tracks[i], tracks[j]
pair = tuple(sorted((t1["track_id"], t2["track_id"])))
b1, b2 = t1["bbox"], t2["bbox"]
c1 = np.array(box_center(b1), dtype=float)
c2 = np.array(box_center(b2), dtype=float)
distance = float(np.linalg.norm(c2 - c1))
mean_diag = (box_diag(b1) + box_diag(b2)) / 2.0
iou = compute_iou(b1, b2)
contact = (
iou >= self.cfg.contact_iou_threshold or
distance <= mean_diag * self.cfg.contact_distance_ratio
)
if contact:
self.pair_streak[pair] += 1
else:
self.pair_streak[pair] = 0
if self.pair_streak[pair] >= self.cfg.persistence_frames:
event = {
"frame": int(frame_idx),
"timestamp_sec": round(frame_idx / fps, 2),
"pair": [int(pair[0]), int(pair[1])]
}
new_events.append(event)
self.events.append(event)
return new_events
